# Scheme Generator

Интерактивный генератор схем (сеток) с возможностью окрашивания ячеек, добавления комментариев, сохранения и экспорта.

## Стек технологий

- **Vue 3** — фронтенд-фреймворк (Composition API, `<script setup>`)
- **TypeScript** — статическая типизация
- **Vite** — сборка и разработка
- **Konva.js + Vue Konva** — рисование и манипуляция 2D-сеткой на canvas
- **PrimeVue** — UI-компоненты (кнопки, модалки, контекстные меню, тултипы, уведомления)
- **@vueuse/core** — composable-утилиты (persistent storage, клавиатуры, даты и т.д.)
- **Tailwind CSS** — утилитарные стили
- **PostCSS + postcss-color-extractor** — обработка стилей и цветов

## Возможности

### Работа со схемой
- Создание сетки с настраиваемой шириной и высотой
- Окрашивание ячеек выбранным цветом (клик по ячейке)
- Режим смещённой сетки (offset) — чередующиеся строки со смещением
- Настраиваемый размер ячеек (50–100px)
- Зум колёсиком мыши и перетаскивание рабочей области

### Комментарии
- Добавление комментариев в любое место рабочей области (правый клик → "Добавить комментарий")
- Перетаскивание комментариев
- Удаление комментариев (правый клик по комментарию → "Удалить")
- Показ/скрытие всех комментариев

### Сохранение и обмен
- Сохранение схем в избранное (localStorage), быстрый доступ через выпадающий список
- Экспорт схемы как PNG-изображение (высокое разрешение, pixelRatio: 3)
- Скачивание схемы как JSON-файла для обмена
- Загрузка схемы из JSON-файла

### Тема
- Переключение тёмной/светлой темы
- Интеграция с prefers-color-scheme

### Горячие клавиши
- `Ctrl/Cmd + S` — открыть модалку сохранения схемы в избранное

### История цветов
- Автоматическое сохранение использованных цветов
- Контекстное меню цвета: использовать как фон / как обводку / удалить из истории

## Структура проекта

```
src/
├── App.vue                     # Корневой компонент (layout: workspace + settings)
├── main.ts                     # Точка входа, регистрация директив и provide
├── constants.ts                # Константы и значения по умолчанию
├── types.ts                    # TypeScript-типы
├── composables/
│   ├── useFileLoader.ts        # Скачивание JSON/PNG, чтение файлов
│   ├── useNotifications.ts     # Toast-уведомления (PrimeVue)
│   ├── useSettings.ts          # Основное состояние приложения (схема, цвета, избранное)
│   └── useWorkspace.ts         # Управление Konva-стадией (зум, курсор, позиция)
├── components/
│   ├── entities/
│   │   ├── workspace.vue       # Konva-холст, рендеринг ячеек и комментариев
│   │   └── settings.vue        # Панель настроек
│   └── modals/
│       └── enter-name.vue      # Модалка ввода названия схемы
├── directives/
│   └── v-focus.ts              # Автофокус на инпуте
├── functions/
│   └── toggleDarkMode.ts       # Переключение темы
└── assets/css/                 # Стили (Tailwind, переменные темы)
```

## Установка и запуск

```bash
# Установка зависимостей
npm install

# Запуск dev-сервера
npm run dev

# Сборка для продакшена
npm run build

# Предпросмотр сборки
npm run preview
```
